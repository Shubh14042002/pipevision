PipeVisionaries

Must have: 
-NODE: version -- 21.x (we used v21.7.3)


Package Installation:
- Open client terminal in editor --> npm install
- Open server terminal in editor --> npm install


Application Startup:
- Open client terminal in editor --> npm start
- Open server terminal in editor --> npm start


Code was made with the following environment:
-Google Chrome
-Client on localhost:3000
-Server on localhost:5000


Docker set up images / container:
- Open base terminal in editor --> docker-compose up --build

Docker run images / container:
- Open base terminal in editor --> docker-compose up
